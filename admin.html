Konversi link sebaran file biasa Google Drive Anda menjadi link visual permanen untuk produk atau katalog.

Langkah Ambil Foto Drive Anda:
  1. Upload foto tumpeng/kue ke folder Google Drive pribadi Anda.
  2. Klik kanan -> pilih Bagikan (Share) -> ubah akses ke "Siapa saja memiliki link" (Anyone with link).
  3. Salin link bagikan, kemudian tempel di kolom konverter di sebelah kanan!
